# AI Video Generation using HeyGen

## 🎥 Demo
[https://youtu.be/OyIcNe70ZpQ]

## ❓ Problem
Creating videos requires time, camera setup, editing, and effort.

## 💡 Solution
Using HeyGen AI, I created videos using AI avatars without recording manually.

## ⚙️ How It Works
1. Write a script
2. Select an AI avatar
3. Generate video using HeyGen
4. Export final output

## 🛠️ Tools Used
- HeyGen (AI video generation)
- ChatGPT (script writing)

## 📊 Use Cases
- Marketing videos
- Training content
- YouTube automation
- Business presentations

## 🚀 Key Learning
AI can significantly reduce video production time and effort.
